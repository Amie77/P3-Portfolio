Interactive Elements

In-text link

CSS selector:

.intext

Sample code:

<a href="#">In-text link</a>

Rendered element:

In-text link

Navigation Link

CSS selector:

.topnav a

Sample code:

<a href="#" >Default Button</a>

Rendered element:

Default Button

Submit/Reset Buttons

CSS selector:

.contact-buttons

Sample code:

<div class="contact-buttons">
 <input type="submit" value="Submit">Submit</button>
 <input type="reset" value="Reset">Reset</button>
</div>

Rendered element:

Text Input Field with Label

Sample code:

<form>
 <label for="name">Name:</label>
 <input type="text" id="name" name="name">
</form>

Rendered element:



Radio Button with Label

Sample code:

<form>
 <input type="radio" id="first-option" name="type" value="first-option">
 <label for="first-option">Label</label>
</form>

Rendered element:

Combined Elements

Main Navigation

CSS selector:

.topnav

Sample code:

<nav class="topnav">
 <div class="topnav-right">
  <a href="#">Home</a>
  <a href="#">Bio</a>
  <a href="#">Projects</a>
  <a href="#">Contact</a>
 </div>
</nav>

Rendered element:

Home Bio Projects Contact
